A test harness for Common Lisp
	 
Description: a test harness allows Common Lisp APIs to be automatically tested.

Author: Kevin Layer, Franz Inc.

This source code has been previously available with Allegro Common
Lisp, and is now being made available to the general public.

Documentation is here:
  http://www.franz.com/support/documentation/current/doc/test-harness.htm

Platform: All ACL versions.
